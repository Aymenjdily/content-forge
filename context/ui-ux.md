# UI/UX Design Concepts

## 1. Dashboard (Home)
- **Hero Section:** Big input box with placeholder: "Enter a topic, paste a URL, or describe what you want to create..."
- **Quick Templates:** Horizontal scroll of template cards (Tech Blog, Viral Thread, Newsletter)
- **Recent Jobs:** Table of recent pipelines with status badges, progress bars, and quick actions

## 2. Job Detail Page (The "Wow" Screen)
- **Visual Pipeline:** Horizontal timeline showing all 7 stages
  - Pending: Gray dashed
  - Running: Animated blue pulse
  - Completed: Green checkmark
  - Failed: Red X with retry button
- **Live Terminal:** Collapsible panel showing real-time logs from Trigger.dev
- **Output Preview:** Tabbed view (Draft | Image | SEO | Scheduled Posts)
- **Actions:** Retry Stage, Cancel Job, Download Assets, Share to LinkedIn

## 3. Content Library
- Grid of generated content cards with thumbnail, title, status, date
- Filters: All | Articles | Threads | Images | Failed
- Search and sort

## 4. Settings
- API key management (OpenAI, Replicate, LinkedIn, Twitter)
- Default preferences (tone, length, auto-schedule)
- Team members (future)
