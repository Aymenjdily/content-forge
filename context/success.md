## Key Learning Opportunities

This project will teach you:
- **Trigger.dev v3** patterns: tasks, orchestration, events, scheduling
- **Next.js App Router** with Server Actions and streaming
- **Prisma** with PostgreSQL for complex relational data
- **AI orchestration** (chaining multiple AI services)
- **Real-time UI** with Server-Sent Events or polling
- **Resilient system design** (retries, idempotency, error handling)
- **Open-source project management** (docs, community, PRs)
